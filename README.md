## 🚀 Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:

- React
- JavaScript
- Radix
- React Icons

## Executando
Após clonar o repositório, acesse a pasta do projeto e execute os comandos abaixo:

```
npm install
npm run dev
```
Acesse http://localhost:5173 para visualizar a aplicação.
